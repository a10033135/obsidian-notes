### ✅ 本月任務彙整
```dataview
TASK
FROM ""
WHERE !completed AND dateformat(start, "yyyy-MM") = dateformat(this.file.day, "yyyy-MM") AND !contains(tags, "#idea")
```

### 🔁 未完成（延續處理）
```dataview
TASK
FROM ""
WHERE !completed AND start < this.file.day AND !contains(tags, "#idea")
```

### 📔 本月日記
```dataview
TABLE WITHOUT ID file.link AS "日期"
FROM -"Weekly" AND -"Monthly" AND -"範本"
WHERE file.day AND dateformat(file.day, "yyyy-MM") = dateformat(this.file.day, "yyyy-MM")
SORT file.day ASC
```

### 📊 本月完成統計
```dataview
TABLE WITHOUT ID
    length(rows) AS "本月完成任務數"
FROM ""
FLATTEN file.tasks AS t
WHERE t.completed AND dateformat(t.completion, "yyyy-MM") = dateformat(this.file.day, "yyyy-MM")
GROUP BY true
```
