租借與購買記錄

GET /api/storyapp/v1/learnings/me/entitlements?page={page}

Request

參數名稱

說明

型別

必填

page

頁碼

String

✓

Response

頂層欄位

欄位名稱

說明

型別

status

請求是否成功

Boolean

data

回傳資料

Object

errors

錯誤訊息列表

Array

pagination

分頁資訊

Object

pagination.current_page

目前頁碼

Integer

pagination.page_items

每頁筆數

預設每頁筆數 20

Integer

pagination.total_pages

總頁數

Integer

pagination.total_count

總筆數

Integer

data.items[] 欄位

欄位名稱

說明

型別

items[].id

entitlement 的 ID

items[].pack_id

項目 ID

Integer

items[].pack_type

項目類型（如 series / packages)

String

items[].title

項目標題

String

items[].thumbnail

項目縮圖網址

String

items[].hint

提示文字（如觀看期限說明）

買斷：不限制觀看期限

尚未開通：請於 … 前開始觀看

租借，已開通：可觀看至 …

租借到期：觀看期已結束

開通過期：開通期已過期

已下架：已下架

String

items[].is_entitlement_avaliable

代表用戶是否擁有權益，尚未過期

包含「已購買未開通」及「已購買已開通」

Boolean

items[].is_navigable

點擊時是否可以導轉到指定 Pack

邏輯：

is_entitlement_available == true or is_offers_saleable == true

用戶權益是已購買且可以開通或可以觀看

或是 對應的 Series/Packages 可以購買

Boolean

items[].label

標籤列表

待開通

Object?

items[].action_button

操作按鈕資訊

Object

items[].labels[] 欄位

欄位名稱

說明

型別

labels[].name

標籤識別碼（如 free）

String

labels[].title

標籤顯示名稱（如「免費」）

String

labels[].bg_color

標籤背景顏色（HEX 色碼）

String

labels[].text_color

標籤文字顏色（HEX 色碼）

String

items[].action_button 欄位

欄位名稱

說明

型別

action_button.title

按鈕顯示文字（如「前往觀看」）

已購買：前往觀看

租借已到期或開通已過期：再買一次

已下架：null

String

action_button.icon

按鈕圖示網址

已下架：Null

其他固定網址，請 yuhsin 上圖

String
