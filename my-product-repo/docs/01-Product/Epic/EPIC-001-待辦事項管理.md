# EPIC-001-待辦事項管理

**來源 PRD：** [[PRD-001-TaskFlow待辦事項管理]]

## 摘要

讓使用者可以建立、檢視、標記完成、刪除待辦事項，作為 TaskFlow 的 MVP 核心功能。

- [x] 使用者可以在畫面上新增一筆待辦事項並立即看到它出現在清單中

- [x] 使用者可以刪除待辦事項，且刪除前有防呆確認
