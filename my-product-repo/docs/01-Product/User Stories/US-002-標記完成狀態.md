# US-002-標記完成狀態

**所屬 Epic：** [[EPIC-001-待辦事項管理]]

## 故事

身為 TaskFlow 使用者，
我想要把已經做完的待辦事項標記為完成（或取消標記），
以便清楚知道哪些事情還沒做。

## 驗收條件 (Acceptance Criteria — Given/When/Then)

- **Given** 清單中有一筆狀態為「未完成」的事項
  **When** 使用者點擊該事項的勾選框
  **Then** 該事項狀態變成「已完成」，並以視覺樣式（刪除線）呈現

- **Given** 清單中有一筆狀態為「已完成」的事項
  **When** 使用者再次點擊該事項的勾選框
  **Then** 該事項狀態變回「未完成」

## 邊界情況 (Edge Cases)

- 快速連續點擊同一筆事項 → 以最後一次的狀態為準，不可讓狀態卡在中間態

## 不在此故事範圍內

- 完成時間戳記顯示於 UI（資料庫有存 `updated_at`，但 v1 UI 不顯示）
