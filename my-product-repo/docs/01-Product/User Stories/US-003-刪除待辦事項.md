# US-003-刪除待辦事項

**所屬 Epic：** [[EPIC-001-待辦事項管理]]

## 故事

身為 TaskFlow 使用者，
我想要刪除不再需要的待辦事項，
以便保持清單的整潔。

## 驗收條件 (Acceptance Criteria — Given/When/Then)

- **Given** 清單中有一筆事項
  **When** 使用者點擊刪除並確認
  **Then** 該事項從清單中移除，且無法復原

- **Given** 使用者點擊刪除
  **When** 使用者在確認對話框中取消
  **Then** 事項保留，不做任何變更

## 邊界情況 (Edge Cases)

- 刪除的同時該事項已被另一個分頁標記完成 → 以刪除為最終結果（後寫入的操作以資源不存在回應）

## 不在此故事範圍內

- 復原/垃圾桶機制（v2 評估）
