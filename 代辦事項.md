# ✅ 代辦事項總覽

> 彙整所有日記／週記／月記裡用 `- [ ] #task` 記錄的待辦事項，不分生活或工作。

## 🔴 逾期未完成
```dataview
TASK
FROM ""
WHERE !completed AND start < date(today) AND !contains(tags, "#idea")
SORT start ASC
```

## 📅 今天
```dataview
TASK
FROM ""
WHERE !completed AND start = date(today) AND !contains(tags, "#idea")
```

## 🔜 尚未排程 / 未來
```dataview
TASK
FROM ""
WHERE !completed AND (!start OR start > date(today)) AND !contains(tags, "#idea")
SORT start ASC
```

## ✅ 最近完成（近 14 天）
```dataview
TASK
FROM ""
WHERE completed AND completion >= date(today) - dur(14 days) AND !contains(tags, "#idea")
SORT completion DESC
```
